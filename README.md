# 项目简介

## 核心功能
- 数据清洗
- 数据预处理
- 数据统计
- 数据可视化
- 研究结论

## 适用场景
- 求职者了解互联网各行业现状
- 企业招聘决策
- 研究人员数据分析

## 技术特点
- 使用numpy进行数据处理
- 利用pandas进行数据预处理
- Flask框架构建后端服务
- Echarts实现数据可视化
